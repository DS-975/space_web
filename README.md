# Space — лендинг о космосе 🌌

Адаптивный лендинг, посвящённый космосу, астрономии и исследованиям Вселенной.  
Свёрстан по методологии **БЭМ** с использованием семантического HTML.

---

## 📋 О проекте

Проект создаётся в рамках интенсивного обучения фронтенд-разработке.  
Цель — получить **чистый, семантичный, адаптивный лендинг** с последующим добавлением JavaScript-интерактива.

---

## 🚀 План разработки (Roadmap)

- [x] **Этап 1. Семантический HTML + БЭМ-нейминг**  
      Создание структуры страницы с правильными тегами и классами по БЭМ.

- [ ] **Этап 2. Базовая стилизация**  
      Подключение шрифтов, CSS-переменные, стилизация шапки и hero-секции.

- [ ] **Этап 3. Flexbox и Grid**  
      Построение сеток для карточек преимуществ и образовательных ресурсов.

- [ ] **Этап 4. Адаптивность (Mobile First)**  
      Медиа-запросы, rem-единицы, адаптация под все устройства.

- [ ] **Этап 5. Анимации и интерактив**  
      CSS-переходы, hover-эффекты, первые скрипты на JavaScript.

- [ ] **Этап 6. Финальная сборка и деплой**  
      Оптимизация, проверка, публикация на GitHub Pages / Vercel.

### `v0.1.0` — БЭМ-нейминг (03.03.2026)

- Переименованы все классы в соответствии с методологией БЭМ
- Классы разделены на блоки, элементы и модификаторы
- Примеры:
  - `.header` (блок) → `.header__logo` (элемент)
  - `.nav` (блок) → `.nav__list`, `.nav__item`, `.nav__link` (элементы)
  - `.nav__link--cta` (модификатор для кнопки регистрации)
  - `.button--primary`, `.button--secondary` (модификаторы кнопок)
- Структура стала чище и подготовлена к стилизации

**Коммит:** `feat: add BEM class names to all blocks`
